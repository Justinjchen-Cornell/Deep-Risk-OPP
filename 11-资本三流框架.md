# 🌊 资本三流框架

> **一句话**：全球资产价格的剧烈波动不是混乱，而是资本三流（总量、方向、流速）被操控的表现。看懂三流，就看清了全球金融战的底层操作系统。
> **什么时候打开**：每天早上看GOR决策卡时、做任何跨境配置决策时、看到美元/美债/黄金异动时。

## 🌊 三流详解

### 一流：总量（Total Volume）

**定义**：全球流动性的总规模——钱的总量在增加还是减少？

| 指标 | 当前值 | 方向 | 含义 |
|------|:---:|:--:|------|
| 美联储资产负债表 | $7.3万亿（从$8.9万亿QT后） | 🔴 收缩 | QT持续抽走流动性 |
| 美国M2增速 | ~2%（从峰值25%回落） | 🔴 收缩 | 货币供应量增速大幅放缓 |
| 美国财政赤字/GDP | 6.7% | 🔴 扩张 | 财政仍在放水，与货币收缩对冲 |
| 全球央行总资产 | ~$30万亿 | 🟡 平稳 | 中国/日本扩表对冲美联储缩表 |
| 美国利息支出 | $1.1万亿/年（19-20%财政收入） | 🔴 恶化 | 利息负担创历史最高 |

**总量对配置的影响**：
- 总量收缩 → 所有资产都面临流动性压力 → 现金为王
- 总量扩张 → 流动性充裕 → 风险资产受益
- 当前状态：**货币收缩 + 财政扩张 = 方向不明确**，但利息支出占20%财政收入的趋势不可持续

### 二流：方向（Direction）

**定义**：资本流向哪里？流入美国 = 向心坍缩，流出美国 = 离心扩散。

| 指标 | 当前值 | 方向 | 含义 |
|------|:---:|:--:|------|
| DXY | 100-102 | 🔴 偏强 | 资本流向美元 |
| 外国持有美债占比 | 20%（从2008年60%下降） | 🔴 长期流出的短期逆转 | 外国央行减持但私人资本增持 |
| 人民币贸易结算占比 | 4%（从3%上升） | 🟢 缓慢上升 | 去美元化的第一步 |
| 人民币储备占比 | 1.95%（从2023Q4的2.29%下降） | 🔴 下降 | 储备货币地位反而在倒退 |
| 黄金储备变化 | 中国增持至2365吨 | 🟢 加速 | 全球央行购金量同比+86% |
| 跨境资本流 | 新兴市场→美国 | 🔴 向心 | 高利率+强美元吸引全球资本 |

**方向对配置的影响**：
- 向心坍缩 → 美元资产受益、新兴市场受损 → 增持美元/美股、减持新兴市场
- 离心扩散 → 美元走弱、新兴市场受益 → 减持美元、增持黄金/新兴市场/A股
- 当前状态：**向心坍缩进行中**，但长期去美元化趋势未被逆转

## 🌀 向心坍缩 vs 离心扩散

### 向心坍缩（Centripetal Collapse）

卢麒元定义：**全球资本被强制拉回美国**的过程——通过制造全球危机（战争/通胀/债务违约）→全球恐慌→资本涌入美元/美债→美元走强→美国可以低价收购全球资产。

```
触发机制: 中东战争→油价暴涨→全球通胀→央行被迫加息→新兴市场崩盘→资本逃向美元
当前阶段: 进行中（油价已涨53%但未突破$150，粮食尚未暴涨）
关键节点: 油粮同时暴涨 = 向心坍缩进入高潮
```

**向心坍缩期间的配置**：
- 美元现金 50-60%
- 黄金 15-20%（被压制的资产，等压制结束会暴涨）
- 油气 15-25%（向心坍缩的"原料"）
- 新兴市场 0-10%
- A股 5-10%（仅在极端低估时配置）

### 离心扩散（Centrifugal Diffusion）

当向心坍缩的"蓄力"阶段结束，美元开始走弱，资本重新流向全球。

```
触发机制: 美联储被迫降息→美元走弱→资本寻找高收益→新兴市场受益
当前阶段: 尚未开始（预计2027.H1）
关键节点: 美元周期见顶（风控日历节点四）
```

## 🧠 Warsh 收割论

### 沃什的"不可能三角"与解决方案

Kevin Warsh（2026年美联储主席提名人）面临的核心矛盾：

```
矛盾: 需要强美元（维持霸权）+ 需要全球买美债（维持融资）+ 需要低利率（减轻利息负担）
现实: 你只能同时拥有其中两个
```

**Warsh的解法**（来自卢麒元分析）：
1. **第一阶段（2026）**：制造全球危机→资本恐慌→美元走强 + 美债需求增加
2. **第二阶段（2027）**：利率下降→置换高息债务→美国财政压力缓解
3. **第三阶段（2028+）**：美元走弱→但此时美国已低价抄底全球资产

**"蓄力—出拳"框架**：要打出重拳，必须先收回手臂（通缩），同时扭转腰部（危机）。恶性通胀的引爆点：**石油和粮食**。油价暴涨+粮价暴涨=通胀冲击=全球紧缩=新兴市场崩盘=向心坍缩完成。
