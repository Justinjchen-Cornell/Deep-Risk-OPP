# 🧭 GOR 方向框架

> **一句话**：用金油比一个指标，决定你今天该配什么方向。
> **什么时候打开**：每天早上看到决策卡时、做任何投资决策前。

---

## ⚡ 30秒速查

```
GOR ≥ 45 → 🔴 原油极端低估 → 70%仓位 → 重仓油气(25%)+黄金(20%)
GOR 30-45 → 🟡 修复周期 → 50%仓位 → 油气(20%)+黄金(15%)+A股(15%)
GOR 20-30 → 🟢 估值均衡 → 30%仓位 → 黄金(15%)+A股(20%)
GOR < 20 → 🟢 原油泡沫 → 10%仓位 → 黄金(7%)+现金(90%)
```

---

## 📐 计算公式

```
GOR(布伦特) = 黄金(美元/盎司) ÷ 布伦特原油(美元/桶)
GOR(WTI)   = 黄金(美元/盎司) ÷ WTI原油(美元/桶)

目标区间: 20-30 (历史均衡)
当前状态: ~52 (2026.06) → 原油极端低估
```

## 🎚️ 风险修正器（仅调仓位，不改方向）

| 条件 | 修正 | 逻辑 |
|------|:---:|------|
| DXY > 99 | -10% | 强美元压制黄金+大宗+新兴市场。DXY受[[04-Token美元进度|Token美元换锚]]进度影响 |
| DXY < 98 | +10% | 弱美元利好全球风险资产 |
| 10Y > 4.3% | -10% | 高利率增加持有成本 |
| 10Y < 4.2% | +10% | 低利率环境利好配置 |
| PBoC 连续2月购金≥2吨 | 黄金仓位锁死 | 央行买盘是底 |
| WTI < $75 | 油气强制降至5% | 硬止损线，不可逾越 |

**最终仓位 = 基础仓位 ± DXY修正 ± 10Y修正**（范围: 5%~80%）

---

## 🔴 硬规则（不可违反）

1. **WTI < $75** → 油气仓位强制≤5%，无论GOR多高
2. **不加杠杆** — 终身铁律
3. **不做空** — GOR框架不适用做空逻辑
4. **不单一品种满仓** — 单资产上限25%
5. **现金≥40%** — [[06-风控日历|SpaceX IPO窗口期]]规则（2026.06起）。风控日历节点优先于GOR方向
6. **分批建仓** — 至少3批，不一次打满
7. **黄金底仓≥15%** — PBoC购金期间锁死
8. **每月复盘** — 对照[[07-决策审计框架#GOR方向判定审计|决策审计]]核查GOR方向准确性

## 📊 历史参考

| 时期 | GOR峰值 | 持续时间 | 回归方式 |
|------|:---:|:---:|------|
| 2020.04 | 60+ | 12个月 | 原油反弹+黄金横盘 |
| 2016.01 | 40+ | 18个月 | 原油慢涨 |
| 2008.12 | 30+ | 24个月 | 危机后原油复苏 |
| 1998.12 | 28+ | 36个月 | 亚洲金融危机后 |

**规律**：GOR从极端位置回归的**方向确定**（原油涨或黄金跌或两者同时），**时间不确定**（12-36个月）。
