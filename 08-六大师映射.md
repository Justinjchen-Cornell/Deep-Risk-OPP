# 🎯 六大师映射框架

> **一句话**：用六位投资大师的独立视角对同一个资产进行交叉验证——任何单一大师都可能看错，但六个人从不同角度得出共识的方向，胜率显著提高。
> **什么时候打开**：GOR指示建仓某方向后、月度六大师报告前、重大事件后需要重新评估持仓。

---

## ⚡ 30秒速查

```
你要分析一个资产 → 六大师并行给出独立判断

大师          核心问题                          信号输出
─────────────────────────────────────────────────────
Buffett      这个公司有护城河吗？安全边际够吗？    bullish/bearish/neutral
Burry        市场忽略了什么风险？数字怎么说？       bullish/bearish/neutral
Taleb        这个资产脆弱吗？尾部风险多大？         bullish/bearish/neutral
Druckenmiller 宏观时机对吗？风险收益不对称吗？      bullish/bearish/neutral
Wood         这是颠覆性创新吗？5年后多大？          bullish/bearish/neutral
Lynch        这个故事能两句话讲清楚吗？             bullish/bearish/neutral
                ↓
        Portfolio Manager 汇总
                ↓
  共识度 ≥ 4/6 bullish → 在GOR仓位上限内积极配置
  共识度 = 3/6 → 标准配置，特别注意反对者的理由
  共识度 ≤ 2/6 bullish → 大幅降低仓位或放弃
```

## 🏛️ 大师一：Warren Buffett — 价值与护城河

### 核心哲学（来自 ai-hedge-fund agent）

| 检查项 | 看什么 | 怎么判断 |
|--------|--------|---------|
| 能力圈 | 你是否理解这家公司的业务？ | 能用两句话讲清楚=在圈内 |
| 竞争护城河 | 毛利率>40%？市场份额>30%？ | 持续高毛利+稳定份额=有护城河 |
| 管理层质量 | ROE>15%持续5年？自由现金流/净利润>0.8？ | 资本配置能力强=好管理层 |
| 财务实力 | 负债/权益<0.5？利息覆盖>5x？ | 低杠杆=能扛过危机 |
| 估值vs内在价值 | PE<历史中位数？FCF收益率>5%？ | 安全边际>0才考虑买入 |
| 长期前景 | 10年后这家公司还在吗？比现在大吗？ | 不会被颠覆=长期持有 |

**信号规则**：
- **Bullish**: 强业务 + 安全边际>0
- **Bearish**: 弱业务 或 明显高估
- **Neutral**: 好业务但安全边际≤0

**在决策框架中的应用**：

| 框架 | Buffett的判断维度 | 权重 |
|------|-----------------|:---:|
| [[02-个股四维研判\|个股四维]] | 基本面维度（估值水位+ROE+现金流） | 40% |
| [[03-接盘论框架\|接盘论]] | 第1项（巴菲特现金占比）的直接信息源 | 信号锚 |
| [[01-GOR方向框架\|GOR方向]] | 不做空、不加杠杆——与Buffett保守主义一致 | 硬规则对齐 |

**何时单独调用Buffett视角**：当你对一只个股的基本面不确定时、当GOR指示重仓但你需要"安全边际验证"时。

## ⚠️ 大师二：Michael Burry — 逆向与数字

### 核心哲学（来自 ai-hedge-fund agent）

| 检查项 | 看什么 | 怎么判断 |
|--------|--------|---------|
| FCF收益率 | 自由现金流/市值 | >10% = 深度价值 |
| EV/EBIT | 企业价值/息税前利润 | <8x = 便宜 |
| 资产负债 | 负债/权益 | >1.5 = 危险 |
| 内部人交易 | 最近3月内部人净买卖 | 净买入 = 正面信号 |
| 催化剂 | 回购/资产出售/分拆 | 有硬催化剂 > 纯等价值回归 |
| 市场情绪 | 媒体/分析师覆盖 | 被厌恶但数字好 = Burry最爱 |

**信号规则**：
- **Bullish**: FCF收益率>10% + 低杠杆 + 内部人买入
- **Bearish**: FCF收益率<3% 或 杠杆过高 或 管理层稀释股东
- **Neutral**: 数字好但无催化剂

**在决策框架中的应用**：

| 框架 | Burry的判断维度 | 权重 |
|------|-----------------|:---:|
| [[02-个股四维研判\|个股四维]] | 基本面+资金面（FCF收益率+内部人交易） | 30% |
| [[03-接盘论框架\|接盘论]] | 第4/5/8项（信用利差+融资余额+机构现金比） | 风险验证 |
| [[05-对冲策略选择\|对冲策略]] | 识别被市场忽视的下行风险→触发对冲 | 二阶机会发现 |

**何时单独调用Burry视角**：当市场一致看多某资产时（逆向检查）、当GOR建议减仓但你犹豫时（确认风险）。

## 🌪️ 大师三：Nassim Taleb — 尾部风险与脆弱性

### 核心哲学（来自 ai-hedge-fund agent）

| 检查项 | 看什么 | 怎么判断 |
|--------|--------|---------|
| 反脆弱性 | 波动对它有利还是有害？ | 能从混乱中获益=反脆弱 |
| 尾部风险 | 最坏情况损失多少？ | 一次极端事件能清零=脆弱 |
| 凸性 | 收益结构是否不对称？ | 涨100%跌10%=凸性 |
| 脆弱性移除 | 杠杆/集中度/流动性风险 | 高杠杆+集中=极度脆弱 |
| 利益绑定 | 内部人持股>10%？ | 管理层与股东利益一致 |
| 波动率机制 | 当前VIX水平 | 低波动=危险（火鸡问题） |

**信号规则**：
- **Bullish**: 反脆弱业务 + 凸性收益 + 不脆弱
- **Bearish**: 脆弱业务（高杠杆/薄利润/波动盈利）或 无利益绑定
- **Neutral**: 信号混合，数据不足以判断脆弱性

**在决策框架中的应用**：

| 框架 | Taleb的判断维度 | 权重 |
|------|-----------------|:---:|
| [[05-对冲策略选择\|对冲策略]] | 整个对冲框架的哲学基础——买保险、不对称收益 | 方法论基石 |
| [[03-接盘论框架\|接盘论]] | 第2/10项（VIX+利差倒挂）——脆弱性积累信号 | 风险预警 |
| [[01-GOR方向框架\|GOR方向]] | 不加杠杆、不做空——硬规则与Taleb哲学一致 | 硬规则对齐 |

**何时单独调用Taleb视角**：VIX<15时（自满检查）、任何"这次不一样"的叙事出现时、建仓后需要评估尾部风险时。

## 📡 大师四：Stanley Druckenmiller — 宏观与动量

### 核心哲学（来自 ai-hedge-fund agent）

| 检查项 | 看什么 | 怎么判断 |
|--------|--------|---------|
| 不对称性 | 上行空间 vs 下行风险 | 上行≥3x下行 = 好交易 |
| 增长动量 | 收入增速在加速还是减速？ | 加速+超预期 = 最强信号 |
| 市场情绪 | 分析师评级+内部人活动 | 支持或矛盾于价格趋势 |
| 资本保全 | 最大回撤控制 | 避免>20%的drawdown |
| 估值容忍 | 愿意为高增长付高估值 | 增速>30% → PE可以>30x |
| 决断力 | 高确信时敢于重仓 | 确信度>70% → 仓位打满 |

**信号规则**：
- **Bullish**: 强增长动量 + 不对称收益 + 支持性催化剂
- **Bearish**: 增长减速 + 风险收益比不利 + 内部人卖出
- **Neutral**: 增长好但估值已定价，或动量弱但基本面好

**在决策框架中的应用**：

| 框架 | Druckenmiller的判断维度 | 权重 |
|------|-----------------|:---:|
| [[01-GOR方向框架\|GOR方向]] | GOR方向判断的宏观逻辑——跨资产资本流动 | 方向确认 |
| [[04-Token美元进度\|Token美元]] | 宏观资本流动分析——美元周期判断 | 周期定位 |
| [[06-风控日历\|风控日历]] | 时间节点中的宏观事件（Jackson Hole/FOMC） | 时机判断 |

**何时单独调用Druckenmiller视角**：GOR方向发生重大变化时、FOMC/央行重大决策前后、需要判断"现在是不是好时机"时。

## 🚀 大师五：Cathie Wood — 颠覆与创新

### 核心哲学（来自 ai-hedge-fund agent）

| 检查项 | 看什么 | 怎么判断 |
|--------|--------|---------|
| 颠覆性创新 | 技术是否在颠覆现有行业？ | 新技术+新商业模式=颠覆 |
| 指数增长 | 收入增速>30%且加速？ | S曲线早期=最大机会 |
| TAM规模 | 可触达市场>1000亿美元？ | 大池子才能养大鱼 |
| 时间视野 | 5年后的世界什么样？ | 不看下季度，看5年 |
| 波动容忍 | 接受高波动换取高回报 | 50%回撤在预期内 |
| R&D投入 | 研发/收入>15%？ | 高研发=未来竞争力 |

**信号规则**：
- **Bullish**: 颠覆性技术 + 指数增长证据 + 大TAM
- **Bearish**: 缺乏真正颠覆性 + 研发不足 + 增长减速
- **Neutral**: 好技术但估值太高 或 技术好但商业化路径不清晰

**在决策框架中的应用**：

| 框架 | Wood的判断维度 | 权重 |
|------|-----------------|:---:|
| [[02-个股四维研判\|个股四维]] | 事件面（S曲线位置+颠覆潜力） | 20% |
| [[04-Token美元进度\|Token美元]] | 算智美元支柱（AI算力控制力） | 信号跟踪 |
| [[01-GOR方向框架\|GOR方向]] | A股科技/创业板配置的参考视角 | 辅助判断 |

**何时单独调用Wood视角**：分析科技/AI/生物技术标的时、Token美元"算智美元"进度出现重大变化时。

## 📊 大师六：Peter Lynch — 草根与常识

### 核心哲学（来自 ai-hedge-fund agent）

| 检查项 | 看什么 | 怎么判断 |
|--------|--------|---------|
| 可理解性 | 业务能用两句话讲清楚吗？ | 讲不清楚=不投 |
| PEG比率 | PE / 盈利增速 | <1 = 低估，>2 = 高估 |
| 现金状况 | 净现金/市值 | >10% = 安全垫 |
| 机构覆盖 | 被多少分析师覆盖？ | 覆盖少=可能有信息差 |
| 分类法 | 慢增长/稳增长/快增长/周期/转机/隐蔽资产 | 不同分类不同操作 |
| 实地调研 | 产品/服务你能亲身体验吗？ | 能体验=有信息优势 |

**信号规则**：
- **Bullish**: 可理解的业务 + PEG<1 + 成长性确认
- **Bearish**: 故事太复杂 或 PEG>2 或 增长幻觉
- **Neutral**: 好公司但PEG在1-2之间

**在决策框架中的应用**：

| 框架 | Lynch的判断维度 | 权重 |
|------|-----------------|:---:|
| [[02-个股四维研判\|个股四维]] | 基本面+资金面（PEG+机构覆盖+股东户数） | 30% |
| [[03-接盘论框架\|接盘论]] | 第3项（IPO狂热）——Lynch最厌恶热门IPO | 信号辅助 |

**何时单独调用Lynch视角**：当你对一只股票"感觉很好但说不清为什么"时、当某标的被媒体狂热追捧时（反向指标）。

## 🎯 六大师共识度矩阵

### 如何计算共识度

```
每个大师对同一资产给出信号: bullish / bearish / neutral
bullish = +1, bearish = -1, neutral = 0

共识度 = (bullish数 - bearish数) / 参与大师数
范围: -1.0 (一致看空) 到 +1.0 (一致看多)
```

### 共识度 → 操作映射

| 共识度 | 含义 | GOR仓位调整 | 操作 |
|:---:|------|:---:|------|
| ≥ +0.67 | 至少5/6看多 | GOR仓位×1.0（满配） | 在方向内积极布局 |
| +0.33 ~ +0.50 | 4/6或3/6看多 | GOR仓位×0.8 | 标准配置，关注反对者理由 |
| -0.17 ~ +0.17 | 势均力敌 | GOR仓位×0.5 | 轻仓试探，等更多信息 |
| ≤ -0.33 | 多数看空 | GOR仓位×0.3或放弃 | 大幅减仓或不碰 |

### 大师冲突时的处理

当Buffett和Wood同时看好但Taleb看空时：
- **Buffett看好** = 公司质量没问题
- **Wood看好** = 长期成长空间没问题
- **Taleb看空** = 短期脆弱性高

→ 结论：好公司+好赛道，但当前时机不好或杠杆太高。**减仓但不放弃**，等脆弱性解除后加仓。

当Burry看多但所有其他人都看空时：
- **Burry看多** = 数字极度便宜，有催化剂
- **其他人看空** = 可能是市场过度悲观

→ 结论：这是Burry最爱的"逆向机会"。**小仓位试探（≤3%）**，等催化剂兑现。
