# 🔍 个股四维研判框架

> **一句话**：任何股票从四个维度交叉验证，至少三个维度通过才考虑建仓。
> **什么时候打开**：你看到一个标的、别人推荐一只票、或GOR框架指示加仓某个方向时。

---

## ⚡ 30秒速查

```
维度一：技术面 → MA排列？支撑/阻力？
维度二：基本面 → PE/PB在什么位置？利润增速？
维度三：资金面 → 北向/主力在买还是卖？ETF资金流？
维度四：事件驱动 → 未来1-3个月有什么催化剂？
        ↓
   四维交叉验证
        ↓
  4/4通过 → 重仓（不超过GOR方向上限）
  3/4通过 → 标准配置
  2/4通过 → 观察仓（≤3%）
  ≤1/4通过 → 不碰
```

---

## 维度一：技术面

| 检查项 | 看什么 | 怎么判断 |
|--------|--------|---------|
| MA排列 | MA5/10/20/60/120 | 多头排列=MA5>MA10>MA20，越整齐越好 |
| 支撑位 | 前低/MA60/MA120 | 现价距最近支撑<5%=安全边际 |
| 阻力位 | 前高/整数关口 | 距阻力>10%=有空间 |
| 成交量 | 量比/换手率 | 放量突破>缩量上涨>放量下跌 |
| MACD | DIF/DEA | 金叉+柱状线放大=买入信号 |

**速判**：MA5>MA10>MA20 且价格在MA20上方 = 技术面OK。反之=观望。

---

## 维度二：基本面

| 检查项 | 看什么 | 怎么判断 |
|--------|--------|---------|
| 估值水位 | PE/PB历史分位 | <30%分位=低估，30-70%=合理，>70%=高估 |
| 利润增速 | 近2季扣非净利润YoY | >20%且加速=优秀，<0%=回避 |
| 毛利率 | 近4季趋势 | 稳定或上升=有定价权，持续下降=竞争恶化 |
| ROE | 近3年均值 | >15%=优质，<8%=一般 |
| 现金流 | 经营现金流/净利润 | >1=利润真实，<0.5=利润存疑 |

**速判**：PE<历史中位数 且 利润增速>15% = 基本面OK。反之=等回调。

---

## 维度三：资金面

| 检查项 | 看什么 | 怎么判断 |
|--------|--------|---------|
| 北向资金 | 沪/深股通近5日净买卖 | 连续净买=外资看好 |
| 主力资金 | 超大单净流入 | 连续3日净流入=主力建仓 |
| 融资余额 | 变化趋势 | 温和上升=散户信心，急剧下降=恐慌 |
| ETF资金流 | 对应ETF份额变化 | 份额增加=被动资金流入 |
| 股东户数 | 季度变化 | 减少=筹码集中，增加=分散 |

**速判**：北向+主力至少一个方向净流入 且 ETF份额未下降 = 资金面OK。

---

## 维度四：事件驱动

| 检查项 | 看什么 | 怎么判断 |
|--------|--------|---------|
| 财报窗口 | 下次财报日期 | 财报前2周=博弈窗口，财报后=确认窗口 |
| 政策催化 | 行业政策/补贴/监管 | 预期中的利好=已定价，超预期的=机会 |
| 机构覆盖 | 最近1月研报数量/目标价 | 研报密集+目标价上调=正向 |
| 解禁压力 | 未来3月解禁量 | >流通盘5%=压力，<1%=无影响 |
| 竞争对手 | 同行季报/新品 | 同行超预期=板块效应，同行暴雷=拖累 |

**速判**：未来3个月至少有1个正催化剂 且 无重大解禁/减持风险 = 事件面OK。

## 📊 决策矩阵

| 通过维度 | 操作 | 仓位上限 | 止盈 | 止损 |
|:---:|------|:---:|:---:|:---:|
| 4/4 | 标准建仓 | [[01-GOR方向框架#30秒速查\|GOR方向上限]] | 30% | -8% |
| 3/4 | 轻仓试探 | GOR方向上限×0.5 | 20% | -5% |
| 2/4 | 观察仓 | 3% | 15% | -3% |
| ≤1/4 | 不碰 | 0% | — | — |

> ⚠️ 如果[[03-接盘论框架#阶段判定\|接盘论]]进入🟠晚期(≥7分)，所有仓位上限再×0.7

## 📋 实战案例：金宏气体（2026.04.24 回顾）

| 维度 | 当时判断 | 判定 |
|------|---------|:---:|
| 技术面 | 股价36.12，MA10约30元，多头排列但有回调压力 | ✅ |
| 基本面 | 氦气三强之一，国产替代逻辑，但估值不便宜 | ✅ |
| 资金面 | 半导体产业链资金关注度高 | ✅ |
| 事件面 | 半导体国产化政策持续，但短期无新催化 | ⚠️ |
| **结论** | 3/4通过 → 建议分批止盈，跌破MA10(30元)清仓 | 轻仓持有 |

## 🔗 深层关联

### 四维研判 → 下一步行动（出链）

| 当四维告诉你… | 去哪个框架 | 做什么 |
|--------------|----------|------|
| 4/4通过，准备重仓 | → [[01-GOR方向框架#30秒速查\|GOR方向]] | 仓位上限=GOR方向的该资产上限 |
| 4/4通过，但接盘论≥7分 | → [[03-接盘论框架#阶段判定\|接盘论]] | 仓位上限再×0.7 |
| 建仓完成，需要保护 | → [[05-对冲策略选择#30秒速查——对冲策略决策树\|对冲-决策树]] | 根据该资产类型选择匹配的对冲策略 |
| 建仓后，检查是否有时间节点冲突 | → [[06-风控日历\|风控日历]] | 确认未来30天无强制减仓窗口 |
| 每月末回顾该股的研判准确性 | → [[07-决策审计框架#个股决策审计\|审计-个股]] | 填入四维评分 vs 实际走势 |

### 其他框架触发 → 回到四维研判（入链）

| 触发框架 | 触发条件 | 回到四维做什么 |
|---------|--------|------------|
| ← [[01-GOR方向框架\|GOR]] | GOR指示加仓某方向(如油气25%) | 用四维框架寻找该方向的具体标的 |
| ← [[06-风控日历\|风控]] | 风控节点要求减仓 | 用四维框架重新评估：哪些标的该先减？ |
| ← [[07-决策审计框架\|审计]] | 审计发现个股连续2次判断失误 | 检查四维中哪个维度在系统性失效 |

### 四维内部的交叉优先级

当四个维度互相矛盾时：**基本面 > 资金面 > 技术面 > 事件面**。基本面决定"该不该买"，资金面决定"市场认不认"，技术面决定"现在买合不合适"，事件面决定"有没有催化剂加速兑现"。
