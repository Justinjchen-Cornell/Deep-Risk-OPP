# 🏛️ 接盘论框架

> **一句话**：机构绑定深度决定救市触发高度——机构在资产里陷得越深，救市来得越快。
> **什么时候打开**：看到美股大跌新闻时、考虑减仓美股时、判断市场顶部时。

---

## ⚡ 30秒速查

```
机构绑定深度越深 → 救市触发点越高 → 跌幅容忍度越低

2000年互联网泡沫: 机构绑定浅 → 跌80%才救 → 纳指从5048跌到1114
2008年次贷危机:   机构绑定中 → 跌50%才救 → 雷曼破产后才出手
2020年新冠危机:   机构绑定深 → 跌20%就救 → 无限QE+直接买企业债

现在(2026): 机构绑定极深 → 跌~15%可能就触发救市？
          但救市的代价在递增 → 每次救市都在透支美元的长期信用
```

---

## 🧠 核心逻辑

### 什么叫"接盘"？

```
市场下跌 → 资产减值 → 机构资产负债表恶化
    ↓
机构持仓深度 = 华尔街+养老金+401k+ETF+保险公司 = 全民利益绑定
    ↓
绑定越深 → 政治压力越大 → 美联储/财政部的救市速度越快
    ↓
但每次救市 = 印钱 = 美元信用透支 = 下一次需要更大的剂量
```

### 三阶段历史验证

| 危机 | 年份 | 机构绑定程度 | 最大跌幅 | 救市触发点 | 救市手段 |
|------|:---:|:---:|:---:|:---:|------|
| 互联网泡沫 | 2000 | 浅（零售为主） | -78% | 跌了2年才降息 | 渐进降息 |
| 次贷危机 | 2008 | 中（投行+银行） | -57% | 雷曼破产后 | TARP+QE1 |
| 新冠危机 | 2020 | 深（全民401k） | -34% | 3周内 | 无限QE+直接买债 |
| 当前周期 | 2026 | 极深（ETF+被动投资） | ? | 预测:-15~-20% | 预测:更激进 |

## 🔟 10个检查项（市场阶段判定）

每项"是"得1分，总分判断市场所处阶段。

| # | 检查项 | 怎么看 |
|:--:|--------|--------|
| 1 | 巴菲特现金占比是否持续上升？ | BRK现金/总资产 >25% = 他在等 |
| 2 | VIX是否持续<15？ | 过度自满 = 顶部信号 |
| 3 | IPO首日涨幅是否>50%？ | 疯狂定价 = 泡沫信号 |
| 4 | 垃圾债与国债利差是否<300bp？ | 信用定价失效 = 风险低估 |
| 5 | 散户融资余额是否创历史新高？ | 杠杆极端 = 脆弱 |
| 6 | meme coin/SPAC是否重新活跃？ | 投机狂热 = 晚期信号 |
| 7 | 美联储是否还在加息/缩表？ | 流动性还在收紧 = 压力在积累 |
| 8 | 机构投资者现金比例是否<3%？ | 满仓 = 没有弹药 = 一跌就踩踏 |
| 9 | 被动投资（ETF+指数基金）占比是否>50%？ | 同质化 = 系统性风险 |
| 10 | 10Y-2Y利差是否倒挂超过12个月？ | 历史上倒挂后必然衰退 |

### 阶段判定

| 得分 | 阶段 | 操作 |
|:---:|------|------|
| 0-3 | 🟢 早期——可以正常配置 | 按[[01-GOR方向框架#30秒速查\|GOR方向]]满配 |
| 4-6 | 🟡 中期——保持警惕 | 仓位降至GOR方向×0.7 |
| 7-9 | 🟠 晚期——防御为主 | 现金≥60%，美股≤10%，对冲升级到[[05-对冲策略选择#策略三\|SPY Straddle]] |
| 10 | 🔴 顶部——随时准备跑 | 现金≥80%，美股清仓 |

### 接盘论 vs 明斯基时刻

接盘论告诉你"跌了有没有人接"，[[06-风控日历\|明斯基时刻]]告诉你"最可能跌的时间窗口"。两者叠加：接盘论≥7分 + 明斯基窗口到来 = 🔴 最高警戒状态。
