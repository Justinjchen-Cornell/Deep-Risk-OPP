# 🛡️ 对冲策略选择框架

> **一句话**：不是"要不要对冲"，而是"用多大成本、对冲哪个级别的风险"。
> **什么时候打开**：GOR发出预警时、VIX异常时、重大事件前、建仓后需要保护时。

---

## ⚡ 30秒速查——对冲策略决策树

```
你要对冲什么？
├── 整个组合的系统性风险 → Straddle SPY（VIX<17时买入）
├── 油价暴跌风险 → WTI $75 Put（已有油气仓位时必配）
├── 地缘政治尾部风险 → 能源ETF长期Call（2-3%组合）
├── 黄金回调风险 → Gold Put + Copper Call 杠铃
├── 人民币贬值风险 → 美元/港币现金+离岸资产
├── A股系统性风险 → 沪深300指数Put（A股>10%时必配）
├── A股黑天鹅风险 → 恒指反向ETF轻仓对冲（备选）
└── 什么都不对冲 → ❌ 裸奔=赌博
```

## 🧠 三阶机会传导链

### 一阶机会（拥挤，放弃）

> 所有人看到同一个事件 → 所有人做同一个交易 → 利润被挤光

**例子**：油价涨了 → 所有人都买油气股 → 油气股已经不便宜了
**操作**：识别一阶，放弃追涨。转向二阶。

### 二阶机会（传导，执行）

> 看事件对相关资产的影响 → 别人还没反应过来的传导链

**例子**：油价涨 → 高耗能行业成本上升 → 化工/航空利润压缩 → 可以做空后者，或买能源ETF+卖航空ETF的对冲组合
**操作**：画出传导链，找到还未定价的环节，建立配对交易。

### 三阶机会（盲区，探索）

> 看事件对市场结构/信用体系/政策方向的长期影响 → 大多数人根本没在想

**例子**：油价持续上涨 → 美联储被迫加息 → 商业地产贷款违约率上升 → 社区银行承压 → 可以做空区域银行ETF
**操作**：这是认知优势的变现。用很小仓位（<2%）押注。

## 📋 四种具体对冲策略

### 策略一：能源ETF长期Call——对冲地缘尾部风险

| 参数 | 说明 |
|------|------|
| 工具 | XLE LEAPS Call（1-2年到期） |
| 仓位 | 组合的 2-3% |
| 执行价 | ATM（平值） |
| 成本 | ~组合的0.5-1%/年 |
| 触发条件 | 中东局势升级、俄罗斯供应中断、关键海峡封锁 |
| 止盈 | 标的价格翻倍时卖出一半，剩余持有 |
| 最大亏损 | 已付出的期权费（已知、可控） |

**为什么不是买原油期货**：期货需要保证金，可能被追保。期权买方只有权利没有义务。

---

### 策略二：WTI $75 Put——保护现有油气仓位

| 参数 | 说明 |
|------|------|
| 工具 | WTI原油 Put |
| 仓位 | 覆盖现有油气仓位的名义金额 |
| 执行价 | $75（GOR系统的硬止损线） |
| 到期 | 3个月（滚动向前） |
| 成本 | <组合的1% |
| 逻辑 | GOR说油价被低估，但如果继续跌到$75以下，你的油气仓位会受伤。Put就是给仓位买保险。 |

---

### 策略三：SPY Straddle——波动率扩张押注

| 参数 | 说明 |
|------|------|
| 工具 | SPY ATM Straddle（买Call+买Put同执行价） |
| 仓位 | 组合的 2-3% |
| 触发条件 | VIX < 17（期权最便宜时） |
| 到期 | 至少3个月以上 |
| 逻辑 | 当波动率处于低位时，市场定价"未来会很平静"。但历史告诉我们平静总是被打破。 |
| 止盈 | 任一方向涨幅覆盖组合成本后卖出一半 |

---

### 策略四：Gold Put + Copper Call 杠铃

| 参数 | 说明 |
|------|------|
| 工具 | Gold OTM Put + Copper OTM Call |
| 仓位 | 组合的 < 2% |
| 逻辑 | GOR极端高位时，要么黄金跌（Put赚钱），要么铜涨（Call赚钱），两者可能同时发生 |
| 不对称性 | 如果GOR从52回归到30，铜Call的弹性远大于黄金Put的弹性 |

---

### 策略五：A股指数期权Put保护 — 为A股7-25%仓位买保险

| 参数 | 说明 |
|------|------|
| 工具 | 沪深300指数期权（IO·CFFEX）或 300ETF期权（510300·上交所） |
| 仓位 | 覆盖A股仓位的名义金额（仅当A股>10%时触发） |
| 执行价 | 沪深300 < 4500 → Put@4200；4500-5000 → Put@4400；>5000 → 直接减仓不做Put |
| 到期 | 3个月（滚动向前） |
| 成本 | A股仓位的 1-2%/季（沪深300期权流动性好，买卖价差窄） |
| 触发条件 | A股仓位≥10% + 沪深300>4500（估值不便宜时需要保护） |
| 止盈 | Put方向涨幅覆盖成本后卖出一半，剩余持有至到期 |
| 最大亏损 | 已付出的期权费 |

**为什么是沪深300而不是个股期权**：
- 你配置的是A股指数ETF（510300）或资源龙头组合，沪深300指数期权直接对冲系统性风险
- 个股期权的流动性远不如指数期权——买卖价差可能吃掉2-3%的成本
- 沪深300指数期权的日均成交额在A股衍生品中最高，滑点最小

**什么时候不买Put**：
- A股仓位<10% → 小仓位不需要保护（现金本身已经是对冲）
- 沪深300<4200 → 极端低估区间，不买Put（GOR框架的"越跌越买"逻辑优于对冲）
- 接盘论≥7分 → 直接减A股仓位到最低，不依赖Put

**A股对冲 vs 直接减仓的决策**：

| 条件 | 操作 | 理由 |
|------|------|------|
| A股≤10% | 不对冲 | 小仓位，现金已是对冲 |
| A股10-15%·沪深300>4500 | 买Put保护 | 保护已有仓位，保留上行空间 |
| A股10-15%·沪深300<4200 | 不加Put·考虑加仓 | 极端低估=买点 |
| A股15-25%·任何估值 | **必须配Put** | GOR框架的A股上限，必须保护 |
| 接盘论≥7分 | 减A股至≤5%·不依赖Put | Put在系统性崩盘中可能因流动性问题无法成交 |

## 🚫 工具红线

| 工具 | 允许？ | 原因 |
|------|:---:|------|
| 买入Call/Put | ✅ | 买方：亏损有限，收益无限 |
| 卖出Call/Put | ❌ | 裸卖空期权：收益有限，亏损无限 |
| 原油期货 | ❌ | 杠杆+追保风险，不适合长期配置 |
| 反向ETF | ⚠️ | 仅限短期（<1月），衰减严重 |
| 恒指期货 | ❌ | 杠杆过高，波动过大 |
| 黄金期货 | ⚠️ | 可替代黄金ETF，但需严格控杠杆 |

## ⏱️ 5步执行清单

```
□ 1. 确认当前GOR区间 + VIX水平 + 是否有重大事件临近
□ 2. 选择匹配的对冲策略（1-4中选1-2个）
□ 3. 计算对冲成本是否 < 组合的3%
□ 4. 设定明确的止盈/止损条件（写在交易单上）
□ 5. 每周回顾：对冲还在保护该保护的风险吗？
```

| 风控节点临近 | 策略组合升级 | → [[06-风控日历\|风控节点]] | 确认节点时间+规则，选择对应对冲 |

### 其他框架触发 → 回到对冲策略（入链）

| 触发框架 | 触发条件 | 回到对冲做什么 |
|---------|--------|-------------|
| ← [[01-GOR方向框架\|GOR]] | GOR建议重仓某方向 | 建仓前先选对冲策略：仓位>15% → 必须配保护 |
| ← [[03-接盘论框架\|接盘论]] | 接盘论升到🟠晚期 | 对冲升级：从单一Put → SPY Straddle |

### 二阶三阶传导链 ↔ 接盘论的联动

传导链的有效性依赖[[03-接盘论框架\|接盘论]]阶段：
- 🟢早期：一阶拥挤，二阶传导链最有价值（市场尚未定价传导效应）
- 🟠晚期：一阶二阶都已失效（所有人都在跑），只剩三阶认知变现
